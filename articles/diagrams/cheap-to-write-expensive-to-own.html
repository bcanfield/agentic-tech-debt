Two costs that used to be one line
What it costs to make a line of code, vs. what it costs to live with it, 2022–2026
Before coding agents the two were the same line, because the person who wrote the code was the person who read it: slow production was the review. Generation then fell to near zero while ownership held, so the lines came apart and a gap opened where nobody pays down. (Curve shapes are illustrative; the crossing is GitClear's 2024 finding that copy-pasted code first beat refactored code.)
